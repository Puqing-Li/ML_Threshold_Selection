# ML Threshold Selection - User Guide (EN)

## Quick Start (GUI)
1. Load Training Data (Step 1)
2. Input Expert Thresholds (Step 2)
3. Input Voxel Sizes (Step 3)
4. Feature Analysis (Step 4, optional)
5. Train Model (Step 5) or Load Last Model (Model & Threshold Config)
6. Load Test Data (Step 6)
7. Predict Analysis (Step 7)
8. Export / Reports, Mean Fabric, Fabric Boxplots (as needed)
